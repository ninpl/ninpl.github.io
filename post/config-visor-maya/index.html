Configuracion de Visor de Maya (Clase Animacion)

Configuracion del visor de Maya

Cada visor, tiene en el menu renderer tiene 3 tipos.

  • Viewport 2.0: Se comporta como lo hace un motor grafico, con el mismo detalle y calidad, es bastante bueno.

Las opciones que tiene el viewport:

Son opciones para el tema de sombras, renderizado, calculo de grafica, efectos de imagen, niebla, nodos y atributos extra.

Ej: Motion Blur crea un desenfoque en diferentes movimientos. Ej: Anti-aliasing suaviza el perfil de los modelos para que no se note la sierra.

  • Legacy Default: Es el por defecto, con niveles normales, normalmente no se trabaja con el en la animacion.
  • Legacy Hight: Es la propuesta para la mayor calidad.
